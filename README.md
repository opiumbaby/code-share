# CodeShare

CodeShare — клиент‑серверное приложение для публикации и обсуждения фрагментов кода. Пользователи могут создавать сниппеты, добавлять теги, собирать подборки в коллекции, оставлять комментарии и сохранять понравившиеся работы в избранное. Проект ориентирован на быстрый обмен знаниями и удобную навигацию по коду.

## Возможности
- Регистрация и вход по email
- Создание и редактирование сниппетов
- Выбор языка программирования из заранее заданного списка
- Теги и фильтрация по тегам/языкам
- Коллекции сниппетов (папки)
- Комментарии и избранное
- Профиль пользователя с активностью

## Технологии
- Next.js (App Router)
- TypeScript
- tRPC
- Drizzle ORM
- Better‑auth
- PostgreSQL

## Установка и запуск

### 1. Установить зависимости
```bash
npm install
```

### 2. Настроить переменные окружения
Создайте `.env` по примеру `.env.example` и укажите:
- `DATABASE_URL`
- `BETTER_AUTH_URL`
- `PORT` (опционально)

### 3. Применить схему базы данных
```bash
npm run db:push
```

### 4. Запустить проект
```bash
npm run dev
```

Откройте в браузере:
```
http://localhost:3000
```

## Скрипты
- `npm run dev` — локальная разработка
- `npm run build` — сборка
- `npm run start` — запуск production
- `npm run db:push` — применение схемы БД

## Структура проекта (кратко)
- `src/app` — страницы и UI
- `src/server` — tRPC‑роутеры и логика сервера
- `src/server/db` — схемы Drizzle
- `src/lib` — клиентские/серверные утилиты
